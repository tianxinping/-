1.算术运算符 + - * / ++ ++i 先加后用;i++ 先用后加 -- 2.比较运算符 真true假false < > <= >= == 等于,只对表面值进行判断，不涉及数据类型 === 绝对等于,还要比较数据类型 != 不等于,只对表面值进行判断，不涉及数据类型 !== 绝对不等于,还要比较数据类型 3.逻辑运算符 && 逻辑与,只有两个数的值都为true时，a&&b的值才为true，否则为false 导盲犬，要都完全不咬人才可以进入考核 || 逻辑或，只有两个数的值都为false时，a||b的值才为false，否则为true 同时为假结果才是假 ! 逻辑非，对a取反，a为true，则结果为false，a为false，则结果为true 4.赋值运算符 = 将右边的值赋值给左边 += 将右边的值加到左边 例如：a+=b 等价于 a=a+b -= 将右边的值减到左边 例如：a-=b 等价于 a=a-b *= 将右边的值乘到左边 例如：a*=b 等价于 a=a*b /= 将右边的值除到左边 例如：a/=b 等价于 a=a/b %= 将右边的值取模到左边 例如：a%=b 等价于 a=a%b 5.条件运算符 条件 ? 表达式1 : 表达式2 当条件为真时，表达式1会被执行；当条件为假时，表达式2会被执行。
