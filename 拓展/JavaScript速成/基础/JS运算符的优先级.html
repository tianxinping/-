圆括号：() 用于分组表达式，改变默认的运算顺序。 成员访问：. 或 [] 用于访问对象的属性或方法。 函数调用：() 用于调用函数。 逻辑非：! 取反操作符，将真变为假，将假变为真。 乘法：*、/ 和 % 用于乘法、除法和取余运算。 加法：+ 和 用于加法和减法运算。 关系运算符：>、<、<= 和>= 用于比较两个值的大小。 相等运算符：== 和 != 用于判断两个值是否相等。 全等运算符：=== 和 !== 用于判断两个值是否完全相等。 逻辑与：&& 当两个条件都为真时返回真，否则返回假。 逻辑或：|| 当至少有一个条件为真时返回真，否则返回假。 条件运算符：? : 用于根据条件选择不同的值。 赋值运算符：=、+=、 =、*=、/= 和 %= 用于给变量赋值。 逗号运算符：, 用于在表达式中连续执行多个操作。 请注意，优先级高的运算符在表达式中会被优先执行，而优先级相同的运算符则会按照从左到右的顺序执行。如果不确定优先级，使用圆括号来明确指定运算的顺序是一个好习惯。 例如，表达式(2 + 3) * 4将首先计算括号内的加法运算，然后再将结果乘以4。最终的结果为20。
