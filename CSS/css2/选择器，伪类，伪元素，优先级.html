    4.
    通配选择器 *
    元素选择器 p h1 h2
    类选择器   .  选择类名为 的元素  class= " ";
    id选择器   #  id= " ";
    交集选择器: 合并 不留空格表示交集，且，共有部分属于交集，在一个盒子内选择
    并集选择器:  ,   隔开表示 或（一般竖着写）
    后代选择器: 空格 表示选择前者的后代
    子代选择器:  >  只选择儿子属性，不选择孙子属性
    相邻兄弟选择器:  + 只选与前者紧紧相邻的兄弟
    通用兄弟选择器:  ~ 选择向下的所有的兄弟
    属性选择器:  []  选择据有某属性(title/class/id)的元素,
        用[]包含在内 例如:[class]
            直接选择某属性 [title="lyric3"] 
            加^表示以某字母开头 [title^="l"]  
            加$表示以某字母结尾 [title$="4"]
            加*表示包含某字母的元素 [title*="a"]
    伪类选择器  
        1,动态伪类：

        元素后加 :link 表示没有访问过的a元素
                :visited 表示访问过的a元素
                前两个为a标签独有
                :hover 选中鼠标悬浮状态的元素
                :active 选中的是激活状态的元素 按下鼠标不松开
                注意顺序 lvha  lv好用
        获取焦点  :focus  表单类元素使用

        2,结构伪类:

        选中的是div的第一个儿子p元素 (按照 所有 的兄弟计算的)
        div>p:first-child { }

        选中的是div的后代p元素,且p的父亲是谁无所谓,但p必须是其父亲的第一个儿子 (按照 所有 的兄弟计算的)
        div p:first-child { }

            选中的是p元素,且p的父亲是谁无所谓,但p必须是其父亲的第一个儿子 (按照 所有 的兄弟计算的)
            p:first-child { }

        选中的div是最后一个儿子p元素 (按照 所有 的兄弟计算的)
        p:last-child { }

        选中的是div的第n个儿子p元素 (按照 所有 的兄弟计算的)
        div>p:nth-child(3) { }
        关于n的值 
            1. 0或者不写 —— 几乎不用。
            2. n : 选中所有子元素 —— 几乎不用。
            3. 1 ~ 正无穷的整数,选中对应序号的子元素。
            4. 2n 或 even : 选中序号为偶数的子元素。
            5. 2n+1 或 odd : 选中序号为奇数的子元素。
            6. -n+3 : 选中前三个 —— 公式 an + b
        
        选中的是div的第一个儿子p元素 (按照所有 同类型 的兄弟计算的)
        div>p:first-of-type { }

        选中的div是最后一个儿子p元素 (按照所有 同类型 的兄弟计算的)
        div>p:last-of-type { }

        选中的是div的第n个儿子p元素 (按照所有 同类型 的兄弟计算的)
        div>p:nth-of-type(5) { }

            nth-of-type他是当前元素的兄弟元素的第n个，而nth-child是当前元素的兄弟节点的第n个当前元素

        选择器优先级（面试高频）

        行内样式 > ID选择器 > 类选择器 > 元素选择器 > * 通配选择器
        同级后来居上
        <h3 class="a" id="b" style="color: aqua;">选择器优先级</h3>

        权重计算（a，b，c）  vscode鼠标放上去会自动计算好
        a: ID选择器个数
        b: 类，伪类，属性 选择器个数
        c: 元素，伪元素 选择器的个数
        比较方法是以此谁高就用谁。如果出现选择器没问题就可以进行排查是不是有属性权重太大了。
            一般来说行类属性可以秒杀，除了 !important  宇宙最强（直接加在属性后面，该属性变为最强,不建议经常使用）
        并集选择器是分开计算的
        div,span,p{ } 权重为(0,0,1)

        has伪类选择器
            div:has(+ h1)  设置h1之前的div样式，通过子元素设置父元素的样式
